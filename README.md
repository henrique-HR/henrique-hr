## 📌 Projetos em destaque

### 🍕 Sistema de Delivery — Pizza Do Ferreira
Sistema completo de delivery com dois painéis distintos: área do cliente e painel do funcionário.

**Funcionalidades:**
- Cardápio interativo com montagem de pizza personalizada (sabores, borda, tamanho)
- Carrinho de compras com cálculo automático
- Checkout com 3 formas de pagamento: Cartão, PIX e na entrega
- Integração com gateway **EFI Bank** para pagamentos automatizados
- Painel administrativo com dashboard financeiro em tempo real
- Gestão de pedidos por status (pendentes, em preparo, despachados, entregues)
- Controle de motoboy e rastreamento de entregas
- Relatórios de vendas por mês e por ano com gráficos interativos

**Tecnologias:** PHP · Framework Sheep · MySQL · JavaScript · Bootstrap · HTML5 · CSS3

🔗 [Ver projeto no ar](https://pizzadoferreira.online)

🤖 Bot WhatsApp com IA — Pizza do Ferreira
Bot de atendimento automático para WhatsApp integrado com inteligência artificial.

⚠️ Status: O bot está funcional, porém ainda não totalmente integrado ao novo sistema da loja, que passou por uma reestruturação recente. A integração completa está em andamento.

Funcionalidades:

Atendimento automatizado 24/7 via WhatsApp
IA conversacional (Groq — LLaMA 3.3 70B) para respostas naturais
Consulta de status de pedido em tempo real
Rate limiter para evitar spam
Simulação de comportamento humano (delay, digitando...)
Webhook HTTP para notificações do painel admin
Histórico de conversa por usuário com limpeza automática

Tecnologias: Node.js · whatsapp-web.js · Groq AI · JavaScript
